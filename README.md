# Chapel of MOOP

A Burning Man theme camp website built with Hugo.

## About

Chapel of MOOP is a Burning Man theme camp celebrating the art and culture of the playa.

## Development

This site is built using [Hugo](https://gohugo.io/) static site generator.

### Local Development

1. Install Hugo (v0.93.2 or later)
2. Clone this repository with submodules:
   ```bash
   git clone --recurse-submodules https://github.com/dmilad/chapelofmoop.git
   ```
3. Run the development server:
   ```bash
   hugo server
   ```
4. Visit http://localhost:1313/

### Deployment

This site is deployed automatically to Netlify when changes are pushed to the main branch.

## Structure

- `/data/` - Content data files (YAML)
- `/layouts/` - HTML templates
- `/static/` - Static assets (images, fonts, etc.)
- `/themes/hugo-story/` - Hugo theme (git submodule)
